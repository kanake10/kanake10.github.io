## Translate Android SDK
*Kotlin · Jetpack Compose · Maven Central · ML Kit*

A lightweight multi-module Android SDK for integrating powerful text translation built on the TranslatePlus API. Supports single text, batch, subtitle, email and HTML translation with an optional Jetpack Compose UI module included out of the box.

- Single text, batch, subtitle, email and HTML translation
- Optional pre-built Compose UI module with full slot API customization
- Optional voice (TTS/STT) and camera (ML Kit OCR) plugins
- Published on Maven Central

[GitHub](https://github.com/kanake10/translate-android) · [Maven Central](https://central.sonatype.com/artifact/io.github.kanake10/translate)

## Fixer — Currency Converter
*Kotlin · Jetpack Compose · MVVM · Clean Architecture · Room · Hilt · . Mockk . Retrofit*

A modular currency converter app built with a strict multi-module architecture. Fetches live conversion rates from the Fixer.io API with full offline support via Room caching.

- Multi-module architecture: `:app`, `:core`, `:libraries:network`, `:libraries:designsystem`, `:features:currency`, `:features:currencyimpl`
- Full offline caching with Room and last-updated timestamp
- Static code analysis with Detekt and Ktlint
- Comprehensive test suite: DAO, Repository, ViewModel, UI and Component tests

[GitHub](https://github.com/kanake10/Fixer)

## Weather App
*Kotlin · Jetpack Compose · MVVM · Retrofit · Room · Hilt*

An Android weather app that fetches current weather and a 5-day forecast from the OpenWeather API with offline storage and graceful error handling.

- City search with current weather and 5-day forecast
- Offline caching with Room and last updated timestamp
- Location-based weather conditions
- Material3 UI with responsive design

[GitHub](https://github.com/kanake10/weather)

## ModernWarfare API
*Ruby on Rails · GraphQL · REST · PostgreSQL*

A REST and GraphQL API for Call of Duty Modern Warfare 2 data — vehicles, operators, missions, maps and guns. Deployed on Render with pagination support.

- REST endpoints for vehicles, operators, missions and maps
- GraphQL endpoint for flexible querying
- Pagination via `will_paginate` gem

[GitHub](https://github.com/kanake10/ModernWarfare) · [Live API](https://modern-warfare.onrender.com)
